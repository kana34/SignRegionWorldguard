# SignRegionWorldguard
Plugin SignRegionWorldguard
Ce plugin permet de vendre des régions Worldguard plus rapidement.

Utilisatiton:

Il vous suffit de poser un panneau sur la région a vendre et d’inscrire sur la première ligne [SRW] ou [SignRegionWorldguard], le prix (2° ligne) et le parent de la region (3° ligne) se remplise grâce au fichier de configuration. Il vous est possible de les modifier lors de la création du panneau.

Commande:

/srw reload  Permet de recharger la configuration

Permission:

signregionworldguard.reload  Permet de recharger la configuration
signregionworldguard.use  Permet d’acheter des régions
signregionworldguard.create  Permet de créer des panneaux

Dépendances:

Vault
Worldguard
Worldedit
